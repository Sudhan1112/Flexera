# Submission Assets

- Dataset file: `submission/founded_dataset.csv`
- Rows: 60
- File type: CSV (supported by Google Form field)

## Power BI (.pbix) Submission Steps

1. Open Power BI Desktop and import `submission/founded_dataset.csv`.
2. Build pages/charts (dashboard, student profile, admin insights).
3. Save file as `FoundEd_PowerBI_Report.pbix`.
4. Upload the `.pbix` file to Google Drive.
5. In Google Drive: Right click file -> Share -> General access -> Anyone with the link -> Viewer.
6. Copy the share link and paste it in the form field: `Power BI report (.pbix)`.
7. For full beginner step-by-step with copy-paste DAX: open `submission/POWERBI_BEGINNER_GUIDE.md`.

## Form Paste Template

- Dataset upload field: upload `submission/founded_dataset.csv`
- Power BI field link: `https://drive.google.com/file/d/<YOUR_FILE_ID>/view?usp=sharing`
