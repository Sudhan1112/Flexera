# FoundEd - Education Analytics

FoundEd is a modern web application for tracking and improving foundational learning in reading and arithmetic, based on ASER-style data.

## Features

- **Teacher Dashboard**: View class performance, charts, and student tables with filters
- **Student Assessment**: Quick 2-minute assessment forms for teachers
- **Student Profiles**: Individual progress tracking with recommendations
- **Admin Dashboard**: District-level analytics and school comparisons
- **AI Recommendations**: Insights based on student data

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- Recharts for data visualization

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) to view the application.

## Project Structure

- `app/`: Next.js app pages
- `components/`: Reusable UI components
- `lib/`: Mock data and utilities

## Build

```bash
npm run build
```
